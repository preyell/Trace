# TRACE

TRACE is an enterprise application that tracks orders, associated financial commitments, and resulting margins, with built-in audit trails, role-based access, and reporting designed to meet operational and compliance requirements.

## Key Features
- Order management
- Finance & additional expense tracking
- Margin computation
- Role-based access control
- Audit trail
- Reports & exports
- Backup & restore support

## Technology Stack
- Java 21
- Spring Boot
- PostgreSQL
- Hibernate / JPA
- JSP

## Build & Run
```bash
mvn clean install
java -jar target/trace.jar
